数据采集工具大概搞出来了
2022-04-08

是什么

一个传统表格文件与网页表单互相转换的工具，且可以依托后端进行数据收集。提供传统表格文件与Web表单的互转，提交传统表单，在线收集数据，在线导出附带数据的传统表格文件。

怎么做

后端提供两个层级的操作，第一层是用户层，第二层是表单层。

用户层可以新增表单，导入表单子用户，删除表单。
表单层接受子用户登录，新增数据，查找数据，更新数据，删除数据，导出数据。

代码库

简单表单 Demo - ( 测试账号 hochenggang@gmail.com 密码 abc123 )

  • 前端 Vue3 + Vite + TypeScript: 基于Naive-ui，自动拉取表单配置，自动配置表单验证。
  • 后端 Python + Fastapi + Docker: 提供前端所需 Api，数据的增删查改以及数据导出。
